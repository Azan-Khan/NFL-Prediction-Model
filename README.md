# NFL-Prediction-Model
Predictive Model trained off of years of NFL statistics per team. 
Dataset From Kaggle: NFL Team Data 2003-2023
URL: https://www.kaggle.com/datasets/nickcantalupa/nfl-team-data-2003-2023

# Model Description (The "How's"):
  All EDA and subsequent code were done in a jupyter notebook utilized in Google Colab. The dataset must be downloaded and uploaded as a csv to your Colab runtime in order for the model to function correctly. The model utilizes running regression in that it trains on data for a few seasons and then uses the next season as test data. Ex. Train: 2013-2018, Test: 2019.

# Strategies
  The model utilizes EDA, bagging, random forest splits...
